# Задание на верстку

## Команды

### `npm start`

Запуск проекта

## Расположение основных файлов

```
app/index.html                целевой html файл для сборки
app/parts                     html компоненты 
app/js/app.js                 js скрипты
app/styles/sass/blocks        стили проекта
app/images/dist               изображения
app/fonts                     шрифты
```
